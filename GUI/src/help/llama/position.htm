Position Function
The position function looks like llama.position(), where llama is the name of a llama on the field. It returns the position of the llama on the field. For instance, Object0.position() will return the position of the llama named Object0. The position is a set of two numbers. The first is the number of pixels in between the left side of the field and the llama. The second is the number of pixels in between the top of the field and the llama.
