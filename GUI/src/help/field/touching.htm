Touching Function
The touching function looks like touching(llama1,llama2), where llama1 and llama2 are 2 llamas on the playing field. It returns a true or false value (also known as a "boolean") stating whether or not llama1 and llama2 are touching. For instance, touching(Object0,Object1) will return whether or not Object0 is touching Object1.
