More Programming
So you've mastered KodeLlama and want to know what else there is to do with computers? We've got a few helpful links to give you an idea of where to start! Disclaimer: We are not associated with the owners of any of these websites or the associated products.
  • Greenfoot: a programming tool that allows you to use Java to program your own games and simulations.
  • Learn Python: an interactive tutorial for Python, a powerful scripting language that's popular amongst programmers
  • Learn Python The Hard Way: a book for a more thorough introduction to Python. Buy the book or use the free html version online.
  • w3schools: a website devoted to tutorials on how to develop websites and web applications using a variety of languages, from HTML to JavaScript to PHP.
