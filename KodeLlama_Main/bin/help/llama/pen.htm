Pen
The pen is the part of the llama that draws on the field. The size of the pen is set using the resize(num) function and can be discovered using the pensize() function. When in the down position, it draws on the field wherever the llama goes. It can be put down using the pd() function and brought up using the pu() function. The pen's color can also be set in the right-click menu of a llama.
