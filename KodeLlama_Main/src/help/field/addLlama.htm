Add Llama Function
The add llama function looks like addLlama(num1,num2), where num1 and num2 are positive numbers like 50 or 83. It creates a llama located num1 pixels from the left side of the field and num2 pixels away from the top of the field. For instance, addLlama(50,50) will create a llama 50 pixels from the left side of the playing field and 50 pixels from the top of the playing field. This is the standard starting position of llama placed using the add button.
