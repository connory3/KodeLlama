Distance Function
The distance function looks like llama1.distance(llama2), where llama1 and llama2 are the names of 2 llama on the field. It returns the distance, in pixels, between llama1 and llama2. For instance, Object0.distance(Object1) will return the distance between the llamas named Object0 and Object1. Object1.distance(Object0) will return the same thing.
